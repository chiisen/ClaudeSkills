Claude Skills 是 Anthropic 在 Claude 上提供的一種「模組化技能系統」，本質上是一組打包好的「給 AI 用的 SOP＋資源＋程式碼」，讓 Claude 能在需要時自動載入，強化處理特定任務的能力，例如文件處理、資料分析或公司內部專屬流程。

## 核心概念說明

Claude 的 Skill（技能）通常實作為一個資料夾，裡面包含：  
- 說明與指令文件（例如 SKILL.md，描述這個 Skill 要做什麼、步驟、限制）
- 可執行的腳本或程式碼（例如 Python）
- 相關資源檔案（如範本、規範、設定檔等）

Claude 在對話中會「動態載入」這些內容，只在判斷有需要時才使用某個 Skill，而不是每次都啟用全部技能。

## 功能與用途

對一般使用者與團隊來說，Claude Skills 可以用來：  
- 把某個專業任務的做法（流程、判斷標準、範本）做成可重複使用的技能，讓 AI 每次都照同一套 SOP 來執行。
- 讓 Claude 更擅長特定型態的工作，例如：  
  - 文件相關（PDF、Excel、Word、PPT）的整理、分析與生成。
  - 依公司品牌規範寫文案或產出報告。
  - 結合程式碼，做資料處理、自動化腳本、甚至產生圖片或 GIF 等工具型任務。

## 和一般 Prompt / 外掛的差異

- 不只是「一段提示詞（prompt）」：Skill 會把流程、規則、資源、程式碼都結構化放在一起，而不是只有文字說明。
- 也不等同傳統「外掛 plugin」：Skills 本身比較像是「程序性知識（how-to）」的打包，必要時還能搭配 MCP 等工具協定，由 MCP 負責連到外部服務，Skill 則教 Claude 要如何用這些工具完成任務。

## 適合你的使用情境（以你目前背景來看）

以你是後端／全端工程師、正在玩 Flask＋Angular／React 的情境，可以想像：  
- 為「撰寫特定風格的技術文件」定義一個 Skill，統一用詞、章節結構與範例格式。  
- 為「分析某類 CSV / MySQL 匯出報表」做一個資料分析 Skill，內建欄位解釋、常用指標計算流程。  
- 為「Code Review 準則」做 Skill，明確寫好檢查面向（效能、SQL、錯誤處理、架構）、回饋格式等，讓 Claude 對你的專案做更一致的 review。
